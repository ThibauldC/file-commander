# File Commander
## Commands:

- `c`: close right screen
- `/`: change path
- `q`: quit

### Left display specific
- `->`: open up right display with directory tree/move to right display
- `shift+->`: open up right display with code
- `n`: new file
- `d`: new directory
- `r`: rename

### Right display specific
- `<-`: move to first display
- `m`: move file/dir from left to right
